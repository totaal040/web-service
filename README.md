# Intro Modal App

Simple Node.js + Express app serving a static page with an introduction screen and a bottom-right "Get Started" button that opens a modal.

## Run locally

1. Install dependencies:

```bash
npm install
```

2. Start the server:

```bash
npm start
```

3. Open the app:

- Visit `http://localhost:3000`

## Structure

- `server.js` – Express server
- `public/index.html` – Intro page markup
- `public/styles.css` – Styling for intro, button and modal
- `public/app.js` – Modal open/close logic
